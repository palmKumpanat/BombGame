ปุ่มสำหรับการเล่นเกม
W A S D ใช้ในการเคลื่อนที่ของตัวละคร
SPACEBAR ใช้ในการยิงกระสุน


1. ทำการ Run ไฟล์ Server.java เพื่อทำการเปิด server 
2. เมื่อเปิดแล้วจะมีข้อความขึ้นว่า Server Activated ใน terminal ให้ทำการ Run ไฟล์ Client.java (ครั้งที่ 1) 
      ผู้เล่นคนที่ 1 จะทำการเชื่อมต่อเข้ามา
3. Run ไฟล์ Client.java อีกครั้ง (ครั้งที่ 2) ผู้เล่นคนที่ 2 จะทำการเชื่อมต่อเข้ามา 
4. แต่โค้ดของเราตั้งค่าให้เมื่อตัวละครทำการเคลื่อนที่จะทำการส่งข้อมูลของ client ให้ server ดังนั้นเมื่อผู้เล่นทั้งสองคนเชื่อมต่อเข้ามา
     อาจจะยังไม่เห็นกัน ดังนั้นให้ทำการเดินตัวละครของทั้งสองจอเพื่อให้ผู้เล่นทั้งสองเห็นกันได้
